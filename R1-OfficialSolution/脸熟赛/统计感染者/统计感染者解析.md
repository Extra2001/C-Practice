# 统计感染者

## 题目介绍
这次的疫请来势汹汹，dch想统计下全国有多少人感染。

他从电视上看到了湖北内和湖北外的确诊数，但是并不会计算全国总共有多少人确诊，你能帮他算出来吗？

## 输入格式
两个正整数，分别表示湖北内和湖北外的确诊数

## 输入样例
```
1000 2000
```

## 输出格式
一个整数，表示全国总确诊数

## 输出样例
```
3000
```

## 神奇的代码
据说下面的代码能够AC
```c
#include<stdio.h>

int main(){
    int a,b;
    scanf("%d %d",&a,&b);
    printf("%d",a+b);
}
```

## 数据范围
绝对不会超过中国的人口总数
