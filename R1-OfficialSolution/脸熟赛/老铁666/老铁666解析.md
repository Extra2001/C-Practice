# 老铁666

| 难度 | 考点               |
| ---- | ------------------ |
| 2    | 基本输入输出、循环 |

## 题目介绍

疫情防控期间，cbd在家无聊，看着“老铁666”的直播，突然想到这样一个算式：

6+66+666+···+ 6···6;          其中最后一个数字含有 *N* 个6；

要求输入非负整数 *N* ，输出上述表达式的结果；如果输入0，则输出0；

愚蠢的cbd不知道怎么算，你能帮帮他嘛？

## 输入格式

一个非负整数 *N*

## 输入样例

```
5
```

## 输出样例

```
74070
```

## 数据范围

0 &le; *N* &le; 9

## 示例代码

```c
#include <stdio.h>
int main(void)
{
	int n;
	int num = 6;
	int total = 0; 
	
	scanf("%d",&n);
	while(n--)  //n减到0时停止循环 
	{
		total += num;
		num = num*10 + 6;
	}
	printf("%d",total);
	
	return 0;
}
```

## 题目分析

考察循环的一道题，用for或者while都可以，更新变量累加。
